auth.user.create(email, password)¶

This function creates a new user if possible.

First, we check if there is a user that exists with the provided email. Otherwise, we encrypt the password with passlib and store the result as the ‘hash’ property in the new user profile. Most of the properties of the User object are random junk left over from old TweetTracker.

Parameters:
  • email – The email of the new user
  • password – The password to use for the new user
Returns:

The new user’s Object_Id or None if failure
